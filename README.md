Telegram Work Time and Salary Bot

Этот скрипт — это бот для Telegram, который помогает автоматически учитывать рабочие часы сотрудников, рассчитывать их зарплату и вести статистику. Он создан на Python с использованием библиотек Telebot и SQLite3, а также использует встроенный модуль datetime.
Основная идея и назначение

Бот предназначен для ситуаций, когда у работника есть часовая ставка, и необходимо точно фиксировать количество отработанных часов, а также считать заработную плату и количество смен.

Это особенно полезно для небольших команд или фрилансеров, которым важно точно учитывать время работы без ошибок.
Основные функции

    Начать работу (Начать) — стартует новую рабочую смену.
    Закончить работу (Закончить) — завершается текущая смена, автоматически высчитывается количество часов, зарплата и обновляется статистика.
    Статистика — показывает вашу персональную статистику (часы, смены, заработанные деньги) или общую статистику по всем сотрудникам.

Статистика включает

    Моя статистика — данные только по текущему пользователю.
    Общая статистика — данные по всем пользователям (для общего обзора).

Встроенная команда

    /sekret — скрытая команда для удаления всей информации из базы данных. Полезно, например, при выплате зарплаты и необходимости сбросить все для следующего счета.

Технологии и зависимости

    Python 3.x
    Библиотека pyTelegramBotAPI (Telebot)
    SQLite3 (встроенная СУБД)
    datetime — для работы со временем

Установка и настройка

    Клонируйте репозиторий

bash

git clone <ваш репозиторий>

    Настройте файл конфигурации config.py
    Создайте файл config.py в корне проекта и добавьте два обязательных параметра:

python

TOKEN = 'ваш_ключ_бота'  # ключ API вашего бота
ADMIN_ID = ваш_айди_админа  # ваш ID в Telegram, чтобы защитить команду удаления базы

    Установите необходимые библиотеки

bash

pip install pyTelegramBotAPI sqlite3

(Для sqlite3 обычно идет в стандартной библиотеке Python, но в некоторых случаях может понадобиться установить отдельно)

    Запустите бота

bash

python bot_script.py

Важные замечания

    Весь сценарий работы предусматривает использование inline-кнопок для взаимодействия.
    Бот хранит все данные в локальной базе SQLite, которая автоматически создается при первом запуске.
    Перед началом работы убедитесь, что у вас настроен правильный TOKEN и ADMIN_ID.

Важное замечание

Постоянно ведите резервные копии базы данных, чтобы избежать потери данных.
Возможные расширения и доработки

    Добавление более детальной статистики (например, по дням).
    Расчет по разным тарифам или проектам.
    Отправка отчетов на email.
    Визуализация данных (графики и диаграммы).

Заключение

Этот бот — ваш помощник для автоматического учета рабочего времени и зарплаты. Он избавит вас от ошибок при подсчёте и сделает прозрачной систему учёта.
